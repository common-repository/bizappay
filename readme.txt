Bizappay for WooCommerce. Accept online payments via Bizappay. Simplyfy your transactions!

== Description ==
This plugin is open-source and free to use and serves to integrate WooCommerce and Bizappay seamlessly. 
Install this plugin to facilitate online payments through Bizappay. Every user of this plugin is required to register and have a valid account with Bizappay.
New user? [Learn More About Bizappay](https://www.bizappay.my/a/MjQ1) and register your Bizappay account now!

This plugin operates by leveraging a Bizappay (third-party service) to enable WooCommerce users to accept payments via Bizappay. 
This integration allows WooCommerce store owners to offer their customers a secure and efficient payment method through Bizappay.
The plugin will transmit certain information to Bizappay, including the buyer's name, phone number, and email address. This information will be utilized by Bizappay 
to facilitate the subsequent processes during the online payment transaction. It is important to note that this plugin does not store any of this information. 
For further details, please visit the [official Bizappay website](https://www.bizappay.my/a/MjQ1).

[Bizappay privacy policy](https://www.bizappay.my/main/privacy-policy/)

By using this plugin, WooCommerce users can enhance their online stores with a reliable and user-friendly payment solution provided by Bizappay, ensuring a smooth 
and trustworthy shopping experience for their customers.

== Installation ==

**Step 1**
– Login to your *Wordpress admin dashboard*
– Navigate to *Plugins* and click *Add new* button.
– Search **Bizappay** and select the plugin that needs to be installed.

**Step 2**
– Activate the plugin through the 'Plugins' screen in WordPress.

**Step 3**
– Navigate to **WooCommerce >> Settings >> Payments**
– Fill up all the required fields 
– Click **Save changes** button

== Frequently Asked Questions ==

= Where can I get the Bizappay's API key? =

Login to your Bizappay account, click on *Settings > API Key*

= Does this plugin and Bizappay support multicurrency? =

Currently this plugin and Bizappay only support Malaysian Ringgit (MYR)

= How can I contact if there are any inquiries regarding the plugin and Bizappay? =

Send us an email to contact@bizappay.my
